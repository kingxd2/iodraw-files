```mermaid
graph TD
    A([开始：用户选择加法]) --> B[提示“请输入一共有几个加数”]
    B --> C[读取 number_of_plus]
    C --> D[调用函数 Plus(number_of_plus)]
    
    subgraph Plus函数内部
        D --> E[提示“请输入加数:”]
        E --> F[初始化数组 arr_plus[number_of_plus]]
        F --> G[设置循环变量 i = 0]
        G --> H{是否 i < number_of_plus?}
        H -- 是 --> I[读取一个加数存入 jiashu]
        I --> J[将 jiashu 存入 arr_plus[i]]
        J --> K[i++]
        K --> H
        H -- 否 --> L[初始化 sum = 0]
        L --> M[设置循环变量 i = 0]
        M --> N{是否 i < number_of_plus?}
        N -- 是 --> O[sum = sum + arr_plus[i]]
        O --> P[i++]
        P --> N
        N -- 否 --> Q[返回 sum]
    end
    
    Q --> R[输出计算结果 sum]
    R --> S([结束])
```
